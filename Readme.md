# 力学基础讲义 README

这是一份面向 CPhO 竞赛学生的基础力学讲义，涵盖运动学基础、静力学基础、牛顿运动定律、功与能、动量与角动量、碰撞专题六个部分，并附有 CPhO 中需要的数学知识。

*本书遵循 CC BY-NC-ND 4.0 协议进行共享。*

- 本书采用 Typst 进行编写；
- 本书的模板基于 ClassicThesis Style for Typst 进行修改；
- 本书的题目很多来自经典竞赛教材《物理学难题集萃》与《物理竞赛专题精编》。

以下是本书的使用说明：

- `main.typ` 为编译主文件，在 typst.app 中选择预览此文件即可；
- `example.typ` 为 ClassicThesis Style for Typst 的原始示例文件，为方便后续其他编者参考，保留此文件，实际不参与书籍的编译；
- `format` 文件夹下是所有的与格式有关的头文件：
  - `lib.typ` 为经过修改的模板主文件；
  - `utils.typ` 为全局定义的为方便书写而设置的公式函数；
  - `watermark.typ` 为水印样式设置。
